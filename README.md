This is a fun and interactive Guess the Number game built using HTML, CSS, and JavaScript.
The user has to guess a randomly generated number between 1 and 100, with feedback given after every guess.
The game also tracks the number of attempts and allows restarting anytime.

Technologies Used:
 HTML
 CSS
 JavaScript

Features:
 Random number generated between 1–100 each round.
 Simple and clean UI styled with CSS.
 Real-time feedback (“Too high”, “Too low”, “Correct!”).
 Restart button to play again.
 Attempts counter.
 Works on all browsers.
